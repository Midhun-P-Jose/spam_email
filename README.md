# Spam Email Detection Project

This project is a Machine Learning model that detects spam messages using SVM and TF-IDF.

## Features
- Text preprocessing
- TF-IDF Vectorization
- SVM classifier


## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy


## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py
